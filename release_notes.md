## 🚀 v0.5.0 版本更新说明

我们正式完成了 Phase 5 阶段，迎来了底层架构和 UI 的大突破！

**✨ 新特性与更新：**
- **Chrome 级多标签页系统**：全新重构的标签页导航栏，支持无缝多开网页，甚至支持双击标签将网页剥离为原生独立窗口。
- **底层浏览器核心升级**：Electron 升级至稳定的 `42.2.0 (Chromium 148)`，性能和安全性大幅提升。
- **自定义站点与油猴脚本引擎**：支持手动配置任意普通 OJ 的题目匹配规则；内置 Tampermonkey 级别的扩展脚本注入引擎。
- **深层 Bug 修复**：完美解决了 Codeforces 教练题等自动重定向情况导致的本地数据库 `canonical_url` 污染问题。
